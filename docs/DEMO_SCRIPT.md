# Demo Script

Target duration: 5-7 minutes.

## 0:00 - 0:20 Project Introduction

GameAsset Forge is an AI 2D game asset production and cloud delivery platform for indie game developers.

## 0:20 - 1:00 Pain Point

Indie developers often lack art resources, while ordinary AI image tools do not produce organized, validated, exportable game asset packages.

## 1:00 - 2:00 Generate Assets

Show the generation page. Enter a platformer game theme, choose pixel art, select character, enemy, item, and tileset, then generate.

## 2:00 - 3:00 Prompt Compiler

Show the generated prompts and explain how natural-language requirements become structured prompts for each asset type.

## 3:00 - 4:00 Asset Library

Show generated assets grouped by category with local paths, upload status, and quality scores.

## 4:00 - 5:00 Quality Inspector

Open a quality report and show score, checks, and issue messages.

## 5:00 - 6:00 Upload and Export

Trigger simulated upload and export the generation package. Show cloud URL, `manifest.json`, and zip path.

## 6:00 - 6:30 Innovation Summary

Emphasize Prompt Compiler, Asset Quality Inspector, and Cloud Asset Hub.
